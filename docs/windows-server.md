# Windows Server 2019

## Introduction

Windows Server 2019 is Microsoft's enterprise server operating system designed for managing networks, hosting applications, providing centralized administration, and supporting enterprise infrastructure. It is widely used in organizations for services such as Active Directory, DNS, DHCP, File Sharing, Remote Desktop Services, and Windows Administration.

In this cybersecurity home lab, Windows Server 2019 was deployed as a virtual machine using VMware Workstation 17 Player. It served as the Windows target system for networking, administration, and security-related activities.

---

# Why Windows Server 2019?

Windows Server plays an important role in enterprise environments, making it an essential operating system for cybersecurity professionals to understand.

Some reasons for selecting Windows Server 2019 include:

- Widely used in enterprise environments
- Provides centralized administration
- Supports Active Directory infrastructure
- Includes built-in security features
- Generates Windows security logs
- Useful for learning Windows administration

Understanding Windows Server is valuable for SOC analysts, penetration testers, incident responders, and system administrators.

# Installation

Windows Server 2019 (Desktop Experience) was installed inside VMware Workstation 17 Player.

The installation process included:

- Creating a new virtual machine
- Attaching the Windows Server 2019 ISO
- Selecting the Desktop Experience edition
- Creating a virtual hard disk
- Completing the Windows installation wizard
- Setting the Administrator password
- Installing system updates

After installation, the server was configured for networking within the isolated cybersecurity home lab.

# Server Manager

Server Manager is the primary administrative console in Windows Server.

It provides centralized access to:

- Local Server Configuration
- Roles and Features
- Event Viewer
- Services
- Performance Monitoring
- Device Management

Server Manager simplifies day-to-day administration by bringing multiple management tools together in a single interface.

---

# Local User Management

Windows Server supports local user and group management through Computer Management.

Administrative tasks include:

- Creating user accounts
- Managing passwords
- Assigning group memberships
- Enabling or disabling accounts
- Managing administrative privileges

Proper user management is essential for maintaining system security.

---

# Windows Firewall

Windows Defender Firewall provides host-based protection by controlling inbound and outbound network traffic.

Firewall rules can be configured to:

- Allow specific services
- Block unwanted connections
- Restrict applications
- Protect network services

Understanding firewall configuration is important when troubleshooting connectivity and securing Windows systems.

---

# Event Viewer

Event Viewer records system, application, and security events generated by Windows.

Common log categories include:

- Application Logs
- Security Logs
- System Logs
- Setup Logs
- Forwarded Events

Event Viewer is an important source of information during incident investigation and security monitoring.

# Remote Desktop Protocol (RDP)

Remote Desktop Protocol (RDP) is a Microsoft protocol that enables administrators to remotely access and manage Windows systems over a network.

Some common uses of RDP include:

- Remote system administration
- Software installation
- Server maintenance
- Troubleshooting
- Enterprise support

Although RDP was not extensively configured in this home lab, understanding its purpose is important because it is one of the most commonly used remote administration protocols in enterprise environments.

# Windows Services

- Windows Update
- Print Spooler

- Remote Procedure Call (RPC)

Understanding Windows Services helps administrators troubleshoot system issues and identify abnormal behavior during security investigations.
